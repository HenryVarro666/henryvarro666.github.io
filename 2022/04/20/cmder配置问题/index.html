配置cmder时遇到的一些问题 // hook

同一行显示不全

输入看着会错乱，但其实是正确的

实际上是中文文件名显示不全

需要环境里面添加设置 set LANG=zh_CN.UTF-8就可以了

(https://github.com/cmderdev/cmder/issues/985)

（可能还要打开monospace）

编码报错

修改编码

环境变量 添加 PYTHONIOENCODING = UTF8

(注意这边不是UTF-8, 而是UTF8)

但是改了这个之后我的picgo失效了。

可能python2，就是电脑默认的那个使用是GBK编码

忽略hook

Any way to disable readline hook warning? #36

不懂这个hook的作用，先码着

路径：C:\ProgramData\Anaconda3\Lib\site-packages\win_unicode_console

修改 readline_hook.py里面

注释掉这个

然后thefuck的官方文档也提到这个报错，可惜没找到合适的解决方法
